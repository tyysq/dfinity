# 下载Demo



使用LinkedUp示例程序来测试调用其它caniste步骤

1. 打开终端切换到ICP示例项目目录
2. 克隆linkedup仓库

```text
git clone git@github.com:dfinity/linkedup.git
```

    3. 切换到linkedup仓库的本地工作目录

```text
cd linkedup
```

    4. 运行下面的命令安装node模块

```text
npm install
```

 如有必要，运行下面的命令修复漏洞

```text
npm audit fix
```

    5. 编辑器中打开dfx.json文件并验证dfx配置里的版本和你安装的dfx版本一致
